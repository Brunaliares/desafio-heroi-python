# desafio-heroi-python

Este projeto cria uma classe genérica que representa um herói de uma aventura. 
O código utiliza:

- Variáveis
- Operadores
- Estruturas de decisão (if/elif/else)
- Laços de repetição (for)
- Funções (métodos)
- Classes e objetos

Permite criar heróis de tipos diferentes (guerreiro, mago, ninja) e simular ataques.
